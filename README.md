# 봉황 메모리즈 (Bonghwang Memories)

> 1988년 아버지의 타임캡슐 - 가이드 없는 셀프 투어 웹앱

## 📋 프로젝트 개요

김해 봉황동을 배경으로 한 몰입형 스토리 체험 웹 애플리케이션입니다.
참가자들이 QR 코드를 스캔하며 5개의 미션을 완료하고, 1988년 아버지가 쓴 위시리스트를 함께 완성하는 감성 투어 프로그램입니다.

### 핵심 컨셉
- **스토리**: 건망증으로 기억을 잃어가는 아버지의 37년 전 위시리스트를 딸 소영이 완성하는 이야기
- **형식**: 가이드 없는 셀프 투어 (배우 없음)
- **진행**: QR 스캔 → 스토리 읽기 → 미션 수행 → 보너스 퀴즈
- **대상**: 가족 단위 참가자

## 🎯 주요 기능

### 1. QR 코드 스캔
- 카메라 활성화 (html5-qrcode 라이브러리)
- 수동 코드 입력 기능
- 미션 코드 검증 (BH001 ~ BH005)

### 2. 미션 진행 관리
- 순차적 미션 잠금/해제
- LocalStorage 기반 진행 상황 저장
- 미션 완료 시 다음 미션 자동 해제

### 3. 스토리 타이핑 효과
- 텍스트가 타자기처럼 천천히 표시
- 스킵 기능 제공
- 50ms 간격으로 한 글자씩 출력

### 4. 보너스 퀴즈
- 4지선다형 퀴즈
- 정답/오답 즉시 피드백
- 보너스 완료 여부 저장

### 5. 진행률 표시
- 프로그레스 바 (0/5 → 5/5)
- 완료한 미션 시각적 표시
- 전체 완료 시 축하 모달

## 💾 데이터 구조

### LocalStorage 저장 데이터
```javascript
{
  completed: [1, 2, 3],        // 완료한 미션 ID 배열
  bonusCompleted: [1, 2],      // 보너스 완료한 미션 ID
  currentMission: 4            // 현재 해제된 미션 번호
}
```

### 미션 데이터 구조 (script.js)
```javascript
missionsData = {
  1: {
    title: "엄마와의 러브스토리",
    location: "봉황1935",
    code: "BH001",              // QR 코드에 들어갈 값
    story: "...",               // 타이핑으로 표시될 스토리
    mission: "...",             // 미션 설명
    bonus: {
      question: "...",
      options: ["1", "2", "3", "4"],
      answer: 1                 // 정답 인덱스 (0~3)
    }
  },
  // ... 2~5번 미션
}
```

## 🔧 주요 함수 설명

### index.html 관련

#### `updateProgressBar()`
- 진행률 바와 미션 카드 상태 업데이트
- 완료된 미션은 초록색으로 표시
- 잠긴 미션은 흐리게 표시

#### `openScanner(missionId)`
- QR 스캐너 모달 열기
- 카메라 권한 요청
- html5-qrcode 초기화

#### `validateCode(code)`
- 입력된 코드 검증
- missionsData에서 일치하는 코드 찾기
- 일치하면 mission.html?id=X 로 이동

### mission.html 관련

#### `loadMissionData(id)`
- URL 파라미터에서 미션 ID 추출
- 해당 미션 데이터 로드
- 헤더, 스토리, 퀴즈 렌더링

#### `typeStory(text)`
- 타이핑 효과 구현
- 50ms 간격으로 한 글자씩
- 자동 스크롤 포함

#### `selectOption(index)`
- 퀴즈 선택지 클릭 처리
- 정답/오답 판정
- 시각적 피드백 (색상 변경)

#### `completeMission()`
- 미션 완료 확인
- LocalStorage 업데이트
- 다음 미션 해제

## 🛠️ 수정 가이드

### 스토리 내용 수정
**파일**: `script.js`
**위치**: `missionsData` 객체 → `story` 속성

```javascript
1: {
  story: `여기에 새로운 스토리 작성
여러 줄로 작성 가능
엔터도 그대로 반영됨`
}
```

### 보너스 퀴즈 수정
**파일**: `script.js`
**위치**: `missionsData` 객체 → `bonus` 속성

```javascript
bonus: {
  question: "새로운 질문?",
  options: [
    "선택지 1",
    "선택지 2",
    "선택지 3",
    "선택지 4"
  ],
  answer: 1  // 0~3 (정답 인덱스)
}
```

### 디자인 색상 변경
**파일**: `style.css`

```css
/* 메인 컬러 변경 */
#D4722B → 원하는 색상 코드

/* 배경색 변경 */
#FFF8DC → 원하는 색상 코드
```

### 타이핑 속도 조절
**파일**: `script.js`
**위치**: `typeStory()` 함수

```javascript
const speed = 50;  // 50ms → 원하는 값 (낮을수록 빠름)
```

## 🐛 알려진 이슈 & 해결

### 이슈 1: QR 스캐너가 작동하지 않음
**원인**: HTTPS가 아닌 HTTP 환경
**해결**: GitHub Pages는 자동 HTTPS 제공 (해결됨)

### 이슈 2: 미션 3 완료 판정 문제
**원인**: 사진 촬영 확인 불가
**해결**: 현장 스태프 배치 또는 자가 체크

### 이슈 3: 타이핑 효과 중 뒤로가기 시 오류
**원인**: setTimeout 미정리
**해결**: `skipTyping()` 함수로 타이머 클리어
